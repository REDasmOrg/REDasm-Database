# REDasm-Database

This repo contains all needed extra information about a Loader or a specific Architecture.<br>
Signatures are generated with [redsigc](https://github.com/REDasmOrg/REDasm-Signature).

### Available Signatures
#### Windows
* Visual C++ 6
* Visual C++ 2003
* Visual C++ 2017 (x86)
* MFC 7.1

#### PlayStation 1
* PsyQ 4.6/7
